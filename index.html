Proyectos Destacados

Tecnologías Utilizadas

Un sitio web interactivo para una organización no gubernamental (ONG) que muestra información sobre sus proyectos, eventos y actividades.

Tecnologías Utilizadas

Una tienda en línea especializada en productos ecológicos y sostenibles.

Tecnologías Utilizadas

  • Frontend: HTML, CSS (con Bootstrap o Tailwind CSS), JavaScript
  • Backend: Django (Python) o Express.js (Node.js)
  • Base de datos: PostgreSQL o MySQL
  • Pasarelas de pago: Integración de Stripe, PayPal u otra pasarela de pago
